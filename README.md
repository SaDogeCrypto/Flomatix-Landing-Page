# Flomatix Resource Hub

This repository hosts the public `flomatix.ai` site for Creative Technology LLC.

The site is intentionally small and stable. It provides legal, support, app association, and platform redirect resources
for apps published by Creative Technology LLC.

## Canonical Routes

- `/`
- `/support`
- `/apps/drop/privacy`
- `/apps/drop/terms`
- `/apps/drop/support`
- `/tiktok-share`
- `/.well-known/apple-app-site-association`

## Deployment

The site is statically exported with Next.js and deployed to GitHub Pages from `main` via `.github/workflows/pages.yml`.

Email for `flomatix.ai` is separate from the website and is handled by Microsoft 365 / Outlook Protection.
